### **Фаза 5. Планирование — для проекта "Мобильное приложение для напоминания о платежах"**

---

#### **1. Определение задач:**

Проект, подробно описанный в **Фазе 3**, необходимо разделить на задачи. Вот список задач для **Фазы 5** (для выполнения проекта по напоминаниям о платежах):

1. **Реализовать экран регистрации по email и паролю.**

   * Проверка формата email и требований к паролю.
   * Запрос к бэкенду на проверку уникальности email.

2. **Реализовать сервис отправки и проверки кода подтверждения по email.**

3. **Разработать экран ввода и подтверждения кода.**

4. **Реализовать экран профиля пользователя (ввод/изменение имени).**

5. **Разработать механизм аутентификации и хранения сессии пользователя.**

6. **Реализовать экран "Мои платежи" с кнопками "Добавить напоминание".**

7. **Разработать экран и логику добавления напоминания о платеже.**

   * Поля: сумма, описание, дата, регулярность, тег.

8. **Реализовать функционал уведомлений о предстоящих платежах.**

9. **Разработать экран "История платежей" для просмотра предыдущих операций.**

10. **Реализовать возможность добавления новых способов оплаты.**

11. **Создать систему тегов для классификации платежей.**

12. **Разработать раздел "Аналитика" для анализа финансовых привычек.**

13. **Реализовать функционал отображения прогресса по финансовым целям (например, "копилка").**

14. **Реализовать систему пуш-уведомлений для напоминания о платежах.**

15. **Разработать функционал для анализа спонтанных трат.**

16. **Реализовать функционал редактирования пользовательских настроек.**

17. **Реализовать функционал настройки регулярных платежей (ежемесячно, ежегодно и т.д.).**

18. **Разработать экран для обучения финансовой грамотности с категориями и статьями.**

19. **Реализовать систему аналитики с показателями доходов и расходов.**
